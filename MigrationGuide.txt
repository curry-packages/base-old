FiniteMap --> Data.Map

The data type changed from "FM" to "Map".
The explicit ordering function is not necessary anymore,
an "Ord" context is needed instead.
The type "Set" in now in Data.Set.
Renames from "Set":

FiniteSet  --> Set
emptySet   --> empty
mkSet      --> fromList
isEmptySet --> null
elementOf  --> member
minusSet   --> difference
setToList  --> toList
union      --> union
